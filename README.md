# StreamerBot Tools by PhaiTing
This is a collection of tools for use in or with StreamerBot.

StreamerBot can be found at:
[streamer.bot](https://streamer.bot/)

## How to import into StreamerBot
I will be saving files to import into StreamerBot with a ".sb" extension. 

You will need to copy the contents of an export file and paste it into 
StreamerBot's "Import String" dialog. You can open this dialog by clicking the "Import"
button at the top of the StreamerBot window.
